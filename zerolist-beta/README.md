# Zerolist

Created with VueJS and Firebase!
Author: Jay Amin

## Build Setup

1. cd into directory
2. npm install
3. cd into functions
4. npm install
5. cd ..
6. npm run dev to test on localhost
  - npm run build to build
  - firebase deploy to deploy (assuming you change all personal values to your project values)
